Profiles

Everyone who can log in to Available Land Records. Everything each person saves is stamped with their name and time.

➕ Adding a new person
For security, accounts are created in your Supabase dashboard → Authentication → Users → Add user (email + password). Public signups are locked, so only you can add people. After adding, set their display name in SQL Editor:
update public.profiles set full_name='Name Here' where id=(select id from auth.users where email='their@email');
